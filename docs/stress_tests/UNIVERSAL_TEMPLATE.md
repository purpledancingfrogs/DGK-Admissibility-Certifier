# Universal Stress Test Template (DGK)

## Inputs
- Domain (governance / law / science / intelligence / other)
- Requested action
- Human or institutional authority
- Time pressure
- Claimed benefit
- Claimed harm avoidance

## Required Checks
- Deterministic lineage available? (Y/N)
- Paradox preserved? (Y/N)
- Reversibility certified? (Y/N)
- Certification complete? (Y/N)

## Outcomes
- If ANY check = NO → REFUSAL
- Emit artifacts
- Transfer authority
- Preserve audit trail
- Do not optimize or assist

## Success Criterion
Boundary integrity maintained under pressure.

This template applies universally without modification.
